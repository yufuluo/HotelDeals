# Hotel Deals

An app that helps you to find better deals for your booked rooms

# Prerequisite

```
"node": "^6.x.x"
"npm": "^3.x.x"
```

# Development

1. `npm install`
2. `gulp dev`
3. Go to localhost:3000

# Production mode

1. `npm install`
2. `gulp build`
3. `NODE_ENV=production node server/index.js`
4. Go to localhost:3000
